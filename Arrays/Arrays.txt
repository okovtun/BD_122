Array - 
1. Динамические и Статические;
2. Как статические, так и динамические массивы бывают
   одномерные, двумерные, трехмерные и т.д. многомерные;

			type name;		//Объявление переменной
			type name[SIZE];//Объявление массива
type - тип элементов массива. Все элементы одного массива могут быть только одного указанного типа.
name - имя массива. Для имнования массивов используются такие же идентификаторы, как и для именования переменных.
SIZE - размер, или количество элементов массива.
